# Flask Image Scraper

This is a Flask application for scraping images from Google search results and storing them in a MongoDB database.

## Requirements

- Python 3.x
- [requirements.txt](requirements.txt) (See file for details)

# Usage
    1 : Run the Flask application:
        -python app.py
    2 : Open your web browser and go to 'http://localhost:5000/' to access the application.
    3 : Enter a search query in the input field and submit the form to scrape images related to the query.

# File Structure
    - app.py: Main Flask application file.
    - requirements.txt: File containing Python dependencies.
    - scrapper.log: Log file for storing application logs.
    - templates/: Directory containing HTML templates.
